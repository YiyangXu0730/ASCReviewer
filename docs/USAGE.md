Start with `docs/SPECIALIZATION_GATE.md`.

This package can support a future dissertation-specific review only after the required specialization inputs are provided.

Do not use this package to infer a marking scheme or score a dissertation without the actual assessment criteria.

For package integrity, compare `artifacts/canonical/ASCREVIEW_FINAL_DELIVERY_PORTABLE_POSIX_20260504_001345.zip` against the SHA256 recorded in `docs/DELIVERY_STATUS.md` and `delivery/TRANSFER_MANIFEST.json`.
