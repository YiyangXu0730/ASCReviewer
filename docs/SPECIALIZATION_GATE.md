# Specialization Gate

A real dissertation-specific ASCReview session may begin only after the user provides:

1. dissertation draft or selected sections
2. actual marking scheme or assignment brief
3. figures/assets and captions where relevant
4. source constraints
5. assessment context
6. supervisor feedback if available

Without those inputs, ASCReview must not:

- score
- grade
- infer teacher preferences
- fabricate rubric criteria
- perform figure critique without assets
- claim dissertation-specific review completion
