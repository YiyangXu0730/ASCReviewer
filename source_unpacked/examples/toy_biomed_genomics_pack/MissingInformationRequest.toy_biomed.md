# Toy Biomedical Genomics Missing Information Request

This fixture supports workflow testing only.

For a real biomedical/genomics review, ASCReview would need:

- actual teacher marking scheme and section weights;
- real draft supplied by the user;
- domain source pack for factual oncology claims;
- dataset accession, release or access date, and repository;
- sample, cell, spot, and feature counts;
- QC/filtering thresholds and rationale;
- normalization, integration, clustering, and annotation methods;
- enrichment database, version, background universe, ranking or threshold strategy, and multiple-testing correction;
- figure files or detailed figure descriptions;
- source data tables, analysis object, and code/notebook;
- target venue or dissertation requirements.

ASCReview should not infer these materials.
