# Figures Failure Modes

This compatibility file points to `figure_legend_failure_modes.md`, the canonical ASCReview figure and legend failure-mode library.

Core failures:

- `FIG-F1`: legend cannot decode the figure.
- `FIG-F2`: panel order does not match the Results narrative.
- `FIG-F3`: labels, colors, symbols, or units are inconsistent.
- `FIG-F4`: statistical notation is unexplained.
- `FIG-F5`: computational plot provenance is missing.

See `patterns/figures/figure_legend_failure_modes.md`.
