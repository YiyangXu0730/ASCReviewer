# Good Figures Patterns

This compatibility file points to `good_figure_legend_patterns.md`, the canonical ASCReview figure and legend pattern library.

Core patterns:

- `FIG-P1`: standalone legend with panel, group, measurement, statistics, symbols, scale, and abbreviations.
- `FIG-P2`: figure panel order matches result narrative order.
- `FIG-P3`: visual encodings use consistent labels, units, colors, symbols, and scales.
- `FIG-P4`: figure generation is reproducible where computational plots are involved.

See `patterns/figures/good_figure_legend_patterns.md`.
