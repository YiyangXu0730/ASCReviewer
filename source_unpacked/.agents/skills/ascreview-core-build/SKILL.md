# ASCReview Core Build


## Purpose

Build the reusable ASCReview workflow core.

## Use when

The task is to create or revise ASCReview's generic workflow, not to review a specific dissertation.

## Required outputs

- Core prompt
- Specialization prompt
- Section review prompt
- Full draft review prompt
- Revision round prompt
- Usage guide


## Evidence rules

- Teacher rubric rules have highest priority.
- Official guidelines and source-derived patterns support secondary evaluation.
- Domain expert heuristics must be labeled and cannot override teacher rules.
- If evidence is missing, ask for it.

## Stop conditions

Stop or mark provisional if:
- marking scheme is missing;
- source provenance is unclear;
- rule becomes too generic to guide review;
- task-specific requirement is being written into reusable core.
